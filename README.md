+     Topic: 校園植物導覽 以臺北市立大學為例
+     Project Introduction :
	  <p>隨著行動科技的發展，具備相機功能的手機已成為孩童隨身常帶的科技產品，也是近年來行動學習(mobile learning)或者終身學習(life-long learning )的有利工具。我們透過QRCODE和AURASMA塑造網頁查詢系統，透過掃描取得資料，直接認識植物。</p>
